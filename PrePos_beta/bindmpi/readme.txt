(note)
This bindmpi is designed to bind all files generated by SolverMPI to
a unified file.


(how to build)
First, create a new directory for the build files and navigate into it:
	mkdir cmbuild
	cd cmbuild
Then,linux:
	cmake ..
	make
     windows:
	cmake -G"NMake Makefiles" -DCMAKE_BUILD_TYPE=Release ..
	nmake
